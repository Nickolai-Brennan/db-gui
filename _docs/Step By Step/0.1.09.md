## Step 0.1.09 — Prettier (root-first, uniform formatting)

Goal: **one formatter** across the whole monorepo, runnable with `pnpm format`.

---

# 0.1.09a) Install Prettier at repo root

```bash
pnpm add -D prettier
```

---

# 0.1.09b) Create `.prettierrc` (root)

Create `/.prettierrc`:

```json
{
  "semi": true,
  "singleQuote": true,
  "trailingComma": "all",
  "printWidth": 100
}
```

---

# 0.1.09c) Create `.prettierignore` (root)

Create `/.prettierignore`:

```txt
**/node_modules/**
**/dist/**
**/.turbo/**
**/.next/**
**/coverage/**
**/.cache/**
**/build/**
pnpm-lock.yaml
```

---

# 0.1.09d) Ensure each package has a `format` script (simple)

You already added `format` in `api` and `shared`. Ensure web has it too.

### Edit `apps/web/package.json` add:

```json
{
  "scripts": {
    "format": "prettier -w ."
  }
}
```

(Keep existing Vite scripts.)

---

# 0.1.09e) Run formatting once

```bash
pnpm format
```

**Output:** repository is consistently formatted.

---

# Done criteria for 0.1.09

✅ `pnpm format` runs and formats all packages
✅ Prettier config applies everywhere
✅ dist/node_modules ignored
